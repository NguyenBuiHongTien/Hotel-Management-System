# Screenshots (optional)

Add PNG or WebP images of your dashboards here for the main `README.md` (e.g. login, receptionist bookings, manager reports).

Suggested filenames:

- `login.png`
- `receptionist-bookings.png`
- `manager-dashboard.png`

Then in the root `README.md` you can embed:

```markdown
![Login](docs/screenshots/login.png)
```

Keeping this folder documents intent even before you add images.
